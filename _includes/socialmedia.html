Use Instead Of: Facebook, Twitter, Instagram, Snapchat.

Facebook / Twitter

Service Description
Mastodon Mastodon is a social network based on open web protocols and free, open-source software. It is decentralized like e-mail. It also has the most users, and the most diverse (in terms of interests) users, looks good, and is easy to setup.
Diaspora* diaspora* is based on three key philosophies: Decentralization, freedom and privacy. It is intended to address privacy concerns related to centralized social networks by allowing users set up their own server (or "pod") to host content; pods can then interact to share status updates, photographs, and other social data.
Friendica Friendica has an emphasis on extensive privacy settings and easy server installation. It aims to federate with as many other social networks as possible. Currently, Friendica users can integrate contacts from Facebook, Twitter, Diaspora, GNU social, App.net, Pump.io and other services in their social streams.

Instagram

Service Description
PixelFed PixelFed is an open source federated alternative to Instagram. This is a photo-oriented social network with filters, comments, likes, shares without the ads and tracking.

YouTube

Service Description
PeerTube PeerTube is an open source alternative to YouTube. While it's still new, it's part of the federated network along with Mastodon and PixelFed, meaning it's growing every day. It's a wonderful, privacy respecting alternative.
