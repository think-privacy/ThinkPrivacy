Use Instead Of: Windows, MacOS.

If you have a choice in your OS, we strongly recommended ditching MacOS or Windows and moving to a Linux driven system. Yet, even with Linux you have choices to make.

Editor's Choice

Name Description Skill Level
Fedora Fedora Workstation is a reliable, user-friendly, and powerful operating system for your laptop or desktop computer. It supports a wide range of developers, from hobbyists and students to professionals in corporate environments. Companies all over the world trust Red Hat Inc. because of their transparency throughout the whole development process. Intermediate
Name Description Skill Level
Tails Tails is a portable OS that you leave installed on a thumbdrive and use when you have access to a computer. This is a pretty secure option and it uses Tor as a default connection to the web, but is not realistic for most users. It is the most secure of the bunch, and is a great option for those who need the utmost security. Advanced
Qubes OS Qubes is an open-source operating system designed to provide strong security for desktop computing. Qubes is based on Xen, the X Window System, and Linux, and can run most Linux applications and utilize most of the Linux drivers. Advanced
Debian Debian is a Unix-like computer operating system and a Linux distribution that is composed entirely of free and open-source software, most of which is under the GNU General Public License, and packaged by a group of individuals known as the Debian project. Advanced

Not sure which Linux distro is right for you? Check out Compute Freely and learn all about the options available to you.

Mobile OS

If you're going to stick with a mainstream OS, we recommend using Apple's iOS over Android. If you are going to use Android, and are comfortable installing your own OS on your device, we recommend using one of the alternative OSes below.

Name Description
GrapheneOS GrapheneOS is a free and open-source security and privacy focused mobile operating system built on top of the Android Open Source Project, only supporting devices offering strong hardware security.
LineageOS A free and open-source operating system for smartphones and tablet computers, based on the Android mobile platform.
Ubuntu Touch Ubuntu Touch is a free and open-source operating system for smartphones and tablets. It's an alternative to the current popular mobile operating systems on the market.
Replicant Replicant is a fully free Android distribution running on several devices, a free software mobile operating system putting the emphasis on freedom and privacy/security
