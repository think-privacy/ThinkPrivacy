Use: Instead of the DNS provided by your ISP.

Name Description
OpenNIC OpenNIC is an alternate network information center/alternative DNS root which lists itself as an alternative to ICANN and its registries. Like all alternative root DNS systems, OpenNIC domains are unreachable to users not using OpenNIC DNS servers.
BlahDNS A small hobby ads block dns project with doh, dot, dnscrypt support.
DNSCrypt A protocol for securing communications between a client and a DNS resolver. The DNSCrypt protocol uses high-speed high-security elliptic-curve cryptography and is very similar to DNSCurve, but focuses on securing communications between a client and its first-level resolver.
